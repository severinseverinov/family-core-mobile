# Egzersiz sesleri

Bu klasördeki WAV dosyaları egzersiz ekranında kullanılır:

- **long_whistle.wav** — Başlangıç / günlük bitiş (uzun düdük)
- **short_whistle.wav** — Egzersiz geçişleri (kısa düdük)
- **tick.wav** — Dinlenme süresinde her saniye tik

Şu an minimal placeholder (sessiz) WAV’lar var. Daha belirgin ses için bu dosyaları kendi MP3/WAV’larınızla değiştirebilirsiniz; dosya adları aynı kalmalı (`long_whistle`, `short_whistle`, `tick`). WAV kullanıyorsanız `.wav`, MP3 kullanıyorsanız `exerciseSounds.ts` içindeki `require` yollarını `.mp3` yapın.
